使用hugo和github搭建博客

   2019-11-02  

安装Hugo 下载hugo 二进制文件 wget https://github.com/gohugoio/hugo/releases/download/v0.58.2/hugo_0.58.2_Linux-64bit.deb 下载完成后执行以下命令: dpkg -i hugo_0.58.2_Linux-64bit.deb 检查安装的hugo版本 hugo version 返回的输出为： Hugo Static Site Generator v0.58.2-253E5FDC linux/amd64 BuildDate: 2019-09-13T08:05:59Z 可以使用hugo –help 命令来查看hugo的帮助手册 hugo --help 返回帮助信息 hugo is the main command, used to build your Hugo site. Hugo is a Fast and Flexible Static Site Generator built with love by spf13 and friends in Go. Complete documentation is available at http://gohugo.io/. Usage: hugo [flags] hugo [command] Available Commands: config Print the site configuration convert Convert your content to different formats deploy Deploy your site to a Cloud provider.
