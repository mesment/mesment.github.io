使用hugo和github搭建博客

   2019-11-02  

安装Hugo

下载hugo 二进制文件

wget https://github.com/gohugoio/hugo/releases/download/v0.58.2/hugo_0.58.2_Linux-64bit.deb

下载完成后执行以下命令:

dpkg -i hugo_0.58.2_Linux-64bit.deb

检查安装的hugo版本

hugo version

返回的输出为：

Hugo Static Site Generator v0.58.2-253E5FDC linux/amd64 BuildDate: 2019-09-13T08:05:59Z

可以使用hugo –help 命令来查看hugo的帮助手册

hugo --help

使用hugo创建网站

可以使用命令hugo new site sitename(www.mesment.com) 来创建一个名叫sitename 的网站

创建网站文件存放目录 这里使用/mnt/hugo
mkdir -p /mnt/hugo
cd /mnt/hugo/
hugo new site blog

执行结果:

root@instance-507hhowz:/mnt/hugo# hugo new site blog
Congratulations! Your new Hugo site is created in /mnt/hugo/blog.

Just a few more steps and you're ready to go:

1. Download a theme into the same-named folder.
   Choose a theme from https://themes.gohugo.io/ or
   create your own with the "hugo new theme <THEMENAME>" command.
2. Perhaps you want to add some content. You can add single files
   with "hugo new <SECTIONNAME>/<FILENAME>.<FORMAT>".
3. Start the built-in live server via "hugo server".

Visit https://gohugo.io/ for quickstart guide and full documentation.

此时在/mnt/data/blog 目录下会多出以下文件和目录

archetypes  config.toml  content  data  layouts  static  themes

在网站目录blog下新建关于页面 about.md

cd /mnt/data/blog
hugo new about.md

返回信息:

root@instance-507hhowz:/mnt/hugo/blog# hugo new about.md
/mnt/hugo/blog/content/about.md created

查看默认生成关于页面信息的内容

root@instance-507hhowz:/mnt/hugo/blog/content# cat about.md 
---
title: "About"
date: 2019-11-02T19:23:49+08:00
draft: true
---

编辑about文件，将内容替换为以下内容:

---
title: "About"
date: 2019-11-02T19:23:49+08:00
draft: false
---
我是mesment,目前从事golang软件开发工作。喜欢陶笛、爬山、健身。

开始准备发布第一篇博客

hugo new post/first.md
